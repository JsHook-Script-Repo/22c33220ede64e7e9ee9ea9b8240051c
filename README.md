# 我的勇者mod

支持所有客户端，国服(所有渠道服)，台服，国际服，东南亚服

## 环境要求

- jshook版本>=1.0.18

- fridamod>=15.2.2.1

## 功能列表：

- 阻止存档，阻止存档后游戏数据不会存到服务器，纯本地，建议硬币和宝石用完再存，不要加太多硬币和宝石，用多少加多少

- 加硬币，可指定数量，最大递增值为999

- 加宝石，只能单人用，最大递增值为99
